Tramplin — это веб-приложение для студентов, выпускников и IT-компаний. 
Платформа позволяет:
- Искать вакансии/стажировки/мероприятия на карте и в списке.
- Фильтровать по тексту и тегам.
- Добавлять вакансии в избранное (для гостей – localStorage, для авторизованных – БД).
- Откликаться на вакансии и отзывать отклики.
- Создавать и редактировать профиль соискателя (ФИО, вуз, навыки, фото, обложка).
- Работодателям – управлять компанией, публиковать вакансии с тегами, сроком действия, координатами.
- Администратору – модерировать компании и вакансии, управлять пользователями.

Технологии: Java 23, Jakarta Servlet 6.0, JSP, JSTL, MySQL 8, Leaflet.js, BCrypt, Gson, Maven, Tomcat 11.

Проект выполнен в рамках конкурса «Прикладное программирование if...else».
